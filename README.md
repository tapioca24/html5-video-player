# html5-video-player

> HTML5 Video Player

## Powered by

- Vue.js
- TypeScript
- Material Icons
- web-animations-js

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
